# Sistemas

Esta carpeta de documentación incluye los siguientes documentos:

1. Informe técnico de Entorno de Ejecución (.pdf): se analiza el entorno y el sistema, y se aportan recomendaciones a la hora de ejecutar y utilizar el programa.

2. Vídeo captura de pantalla de ejecución (.mp4): muestra un proceso normal de ejecución del programa, con las 4 acciones del CRUD, el filtrado y la exportación al archivo .xml.

3. Esquema simple del sistema (.png): incluido también en el informe técnico, este esquema muestra brevemente los 3 apartados principales del programa (interfaz gráfica, lógica y bases de datos).
